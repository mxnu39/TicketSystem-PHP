<h1>TicketSystem-PHP</h1>

<p>
Dieses Projekt ist ein webbasiertes Ticket-System, das mit <strong>PHP</strong> und einer
<strong>Datenbank</strong> arbeitet. Tickets werden automatisch über einen Bot erstellt
und anschließend auf einer Website übersichtlich und grafisch dargestellt.
</p>

<p>
Der Fokus liegt auf der Auswertung und Visualisierung von Ticket-Daten, wie z.B.
offene Tickets, Ticket-Anzahl pro Status oder allgemeine Statistiken.
</p>

<h2>Funktionen</h2>

<ul>
  <li>Automatisches Erstellen von Tickets über einen Bot</li>
  <li>Auslesen von Ticket-Daten aus einer Datenbank</li>
  <li>Grafische Darstellung von Ticket-Statistiken</li>
  <li>Übersicht über offene, geschlossene und aktive Tickets</li>
  <li>Anzeige der Anzahl von Nachrichten/Einträgen pro Ticket</li>
</ul>

<h2>Technologien</h2>

<ul>
  <li>PHP</li>
  <li>MySQL / MariaDB</li>
  <li>HTML / CSS</li>
  <li>JavaScript (für Diagramme & Interaktivität)</li>
</ul>

<h2>Ziel</h2>

<p>
Ziel dieses Projekts ist es, Ticket-Daten verständlich und visuell aufzubereiten,
um einen schnellen Überblick über den aktuellen Support-Status zu ermöglichen.
</p>

<h2>Status</h2>

<p>
Das Projekt befindet sich in aktiver Entwicklung und kann jederzeit erweitert
oder angepasst werden.
</p>
